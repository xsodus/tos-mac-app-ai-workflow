# Launch App Via Spotlight

## When To Use

Use this skill when the goal is to open an installed macOS app by name from the keyboard.

## Workflow

1. Open Spotlight with `Cmd+Space`.
2. Type a distinctive part of the app name.
3. If Spotlight shows multiple results, use arrow keys to select the app entry rather than a file or web suggestion.
4. Press `Return` to launch the app.
5. Verify the app window appears and is frontmost.
6. If Spotlight shows no app match, refine the query with more of the app name.

## Notes

- Prefer this workflow for fast app launching when the app is already installed.
- Keep the search query as short as needed to disambiguate the desired app.
- Use the app title and window appearance as confirmation that launch succeeded.
